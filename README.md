# rutherford_project


GLAD в связке с GLFW используется для загрузки и инициализации
указателей на функции OpenGL во время выполнения. Поскольку
функции OpenGL различаются в зависимости от ОС и драйверов 
видеокарты, GLAD (GL Loader) находит адреса этих функций, 
позволяя использовать их в коде, а GLFW создает окно и контекст.
